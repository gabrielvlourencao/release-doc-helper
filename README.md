# Release Doc Helper

Sistema de gerenciamento e acompanhamento de documentos de release.

## Estrutura do Projeto

```
release-doc-helper/
├── release-helper-app/    # Aplicação Angular
│   └── README.md          # Documentação detalhada
└── releases/              # Exemplos de documentos de release
    └── release_DMND0011870.md
```

## Quick Start

```bash
cd release-helper-app
npm install
npm start
```

Acesse: http://localhost:4200
